# AGENTS instructions for this repository

- This repository is for **simulation-only optical neural network research**.
- Always run tests after major code changes.
- Always save experiment outputs under `outputs/`.
- Always update `docs/progress/` after meaningful experiments.
- Prefer small pilot runs before launching large sweeps.
- Topic priority is: **Topic 1 first**, then Topic 2, then Topic 4.
- New experiments must include: config file, runnable entrypoint, and output summary.
- Avoid unnecessary dependency bloat.
- Keep code scientific, modular, deterministic where feasible, and reproducible.
